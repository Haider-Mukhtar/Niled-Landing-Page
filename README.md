A Landing Page using React, TypeScript and TailwindCSS.

## Run Locally

Clone the project

```bash
  git clone https://github.com/Haider-Mukhtar/Niled-Landing-Page.git
```

Go to the project directory

```bash
  cd Niled-Landing-Page
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
